MAP Testing

Second Grade and Up

  • Students cannot pass or fail this test.
  • This test will not affect grades.
  • It’s normal for students to only answer about half the questions correctly.
  • Please don't help your student understand or answer test questions.
  • These results will help us know what students are ready to learn.

Test Preparation

Please be available to support your child’s during testing.
If you will not be available, please let your teacher know that now.

Before Test Day

  1. Make sure you have:
    • a compatible device (using the list below),
    • headphones, and
    • an internet connection.
  2. Check Device Readiness
    using the link above or go to check.nwea.org
  3. Watch the introduction video below.
  4. Practice the test at practice.nwea.org
    Username: grow
    Password: grow

On Test Day

  1. Provide a quiet space for testing and reduce distractions for your student as much as possible.
  2. Join your teacher’s Zoom link and wait for the teacher to open the testing session.
  3. Use the Session Name and Password to log in at test.mapnwea.org/#/nopopup and wait for the teacher to confirm.

Compatible Devices

Chromebook

  • Operating System
    • Chrome OS 78 or higher is recommended
    • Chrome OS 65 or higher will probably work
  • Internet Browser
    • Chrome

Windows PC

  • Operating System
    • Windows 7
    • Windows 8.1
    • Windows 10
  • Internet Browser
    • Chrome
    • Firefox
    • Edge

Mac

  • Operating System
    • MacOS 10.13 or higher is recommended
    • MacOS 10.12 or higher will probably work
  • Internet Browser
    • Chrome
    • Firefox
    • Safari

iPad

Using a regular computer is easier than using an iPad because you need to have Zoom and the Internet Browser running at the same time, but you might be able to make an iPad work, especially if you can use the Internet Browser on the iPad and another device for Zoom.

  • Operating System
    • iOS 12 or higher is recommended
    • iOS 9 or higher will probably work
  • Internet Browser
    • Safari
    • NWEA Secure Testing (not compatible with iPad mini)
